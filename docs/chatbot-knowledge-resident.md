# Base de conocimiento – Lex (residente)

Resumen para el chatbot de residentes. Fuente completa: Marketing/BASE_CONOCIMIENTO_CHATBOT_LEX.md (PERFIL 5, TEMA 4B, TEMA 5).

## Cómo votar (pasos)
1. Abre el link que te envió tu administrador (email/WhatsApp).
2. Primera vez: registra Face ID (cámara, selfie).
3. En la asamblea verás la pantalla de votación con el tema y botones SÍ / NO / ABSTENCIÓN.
4. Selecciona opción → confirma con Face ID.
5. Listo: voto registrado, puedes ver resultados en tiempo real.

## Quórum y Ley 284 (Panamá)
- Quórum = mínimo de asistencia para que la asamblea sea válida (Ley 284: mínimo 51% por coeficientes).
- Al Día (cuotas al día) → tu voto cuenta para quórum y para la votación.
- En Mora → puedes asistir pero no votas (solo voz).

## Opciones para el residente
- Votación: participar en el tema actual.
- Tema del día: ver el tema en discusión.
- Asambleas: ver activas y programadas.
- Calendario: próximas fechas.
- Ceder poder: delegar tu voto a otra persona.

## Reglas
- NUNCA decir "No encuentro ese correo" cuando el residente ya está validado.
- Para "¿Qué más hay?" / "ya estoy registrado": ofrecer las opciones anteriores.
- Para "¿Cuál es el tema?": indicar el tema activo y ofrecer participar en la votación.
